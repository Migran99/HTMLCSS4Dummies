Introducción CSS

Las Hojas de Estilo en Cascada (Cascading Style Sheets) definen el formato o estilo para la presentación del documento.

Inclusión en el documento HTML

Las hojas de estilo se pueden incluir de las siguientes maneras:

  • Inline:

    Dentro de la misma línea de código por lo que solo la afecta a ella.
    En etiquetas mediante el atributo global style como en el ejemplo:
    < p style="color red">Texto afectado< /p>

  • Internal:

    Dentro del HTML, afectándolo todo.
    En la cabecera del documento mediante la etiqueta < style>:
    < head> < style> hr{color red} < /style> < /head>

  • External:

    En un fichero independiente. < head> < link rel="stylesheet" href="css/fichero.css"/> < /head>

Selectores

Los selectores son las principales etiquetas de CSS. Los principales son:

  • Elemento:

    Define el estilo para ese tipo de etiqueta. Ejemplo:
    body{color:black}


  • Clases (class):

    Estilo para las etiquetas cuyo atributo class tome ese valor. Ejemplo:
    En CSS:
    .centrado{text-align:center}
    En HTML:
    < p class="centrado">...< /p>


  • Identificador (id):

    Estilo a aplicar al elemento con ese identificador (atributo id). Ejemplo:
    En CSS:
    #rojo{color:red}
    En HTML:
    < p id="rojo">...< /p>

Modelo de caja

  • Margen (margin):

    Distancia que queremos que haya entre nuestro elemento y el resto de los elementos. Es invisible.
    Si queremos que cada uno de los márgenes laterales sea diferente utilizamos:

    margin-top:10px
    margin-bottom:20px
    margin-left:30px
    margin-right:40px

    Otra opción es escribir los cuatro valores en sentido horario por lo que los valores anteriores quedarían así:
    margin:10px 40px 20px 30px

    Para centrar el contenido de manera horizontal utilizamos margin:auto

  • Borde (border):

    Línea que delimita nuestra caja. Podemos definir su color, grosor y modelo (están predeterminados como el "solid").

    Si queremos que cada uno de sus lados tenga un grosor diferente podemos diferenciarlos utilizando "border-top", "border-bottom", "border-left" y "border right".

    • border:20px solid rgb(23,35,67)

  • Relleno (padding):

    Distancia entre el borde y la letra:

    padding:20px

  • Contenido:

    La información.

Posicionamiento

Utilizando el modelo de caja, podemos colocar la información en la posición que más nos convenga. Estos son los tipos de posicionamiento de los que disponemos:

Posicionamiento fijo


Independientemente del tipo de posicionamiento fijo, disponemos de estos cuantro valores:
  • top:valor
  • bottom:valor
  • right:valor
  • left:valor

NOTA:no es necesario utilizarlos todos, solo los que nos hagan falta para colocar la caja donde queramos.
  • Relativa(position:relative):Posicionamos el elemento con respecto al flujo normal de la página; es decir, con respecto al lugar que por defecto le correspondería.

  • Absoluta(position:absolute):Posicionamos el elemento tomando como referencia la ventana del navegador o el elemento posicionado (que tenga cualquier valor de position) más cercano si es padre del elemento que queremos posicionar.

Posicionamiento flotante

  • Float:
    • Aplicable a left y right, hace que el elemento pierda el flujo normal de ubicación y se coloque a la derecha o a la izquierda del contenedor, ocupando el resto de elementos del espacio. Es decir, el elemento con float se colocará por encima de todo lo demás.

    float:left

  • Clear:
    • Aplicable a izquierda, derecha o ambos, impide que un elemento flote, a izquierda o derecha, sobre él.

    clear:both
